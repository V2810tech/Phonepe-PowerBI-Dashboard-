# 📊 PhonePe Transaction Analysis Dashboard

## 📌 Project Overview
This project is an interactive Power BI dashboard that analyzes PhonePe transaction data.

## 🚀 Features
- Interactive Date Slicer
- KPI Cards
- Money Transfer Dashboard
- Recharge & Bills Dashboard
- Insurance Dashboard
- Loans Dashboard
- Monthly Trend Analysis
- Payment Status Analysis

## 🛠 Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## 📈 Skills Demonstrated
- Data Cleaning
- Data Visualization
- Dashboard Design
- Business Intelligence
- Data Analysis
